  <action>
Create two new Typst module files:

**branding.typ** — Exports `render-logo(logo-path)` and `render-footer(footer-text)` functions:

`render-logo(logo-path)`:
- If logo-path is none or empty string, return (render nothing)
- Unescape underscores: `let clean-path = logo-path.replace("\\", "")` (Pandoc escapes `_` to `\_` in YAML metadata — confirmed pitfall from research)
- Render: `align(right)[#image(clean-path, width: 2.5cm)]` + `v(0.5em)`
- Logo is placed at the top of the page, before the title block

`render-footer(footer-text)`:
- Returns a Typst content value for use in `set page(footer: ...)`
- If footer-text is none, return `none`
- Otherwise: `align(center)[#text(size: 8pt)[#footer-text]]`

Note: `apply-base-styles` in base.typ already handles primary-font, title-font, margins, accent-color via `set` rules. branding.typ handles the content-producing branding elements (logo, footer).

The 9 branding hooks map to modules as follows:
1. logo -> branding.typ `render-logo()`
2. primary-font -> base.typ `set text(font:)` (already in Plan 01)
3. title-font -> base.typ (title font applied in titleblock.typ if provided)
4. margins -> base.typ `set page(margin:)` (already in Plan 01)
5. title-page-style -> controlled by format overlay (which sections to show)
6. accent-color -> base.typ `show heading:` rule (already in Plan 01)
7. report-number-block -> titleblock.typ (already in Plan 01)
8. footer -> branding.typ `render-footer()`, applied in base.typ
9. disclaimer-page -> disclaimer.typ

**disclaimer.typ** — Exports `render-disclaimer(text, position, enabled)` function:
- `text` defaults to `"The views expressed in this paper are those of the authors and do not necessarily reflect those of their affiliated institutions."`
- `position` defaults to `"last"` (either `"first"` or `"last"`)
- `enabled` defaults to `false`
- If not enabled, return (render nothing)
- If enabled: `pagebreak()` + `set page(numbering: none)` scoped in a block + `align(center + horizon)[#block(width: 80%)[#set text(size: 10pt, style: "italic")` + the text content + `]]`
- Note: position is handled by the caller (format overlay) — disclaimer.typ just renders the page; the format overlay calls it before or after body based on position

Also create a simple figure/table caption show rule. Add to base.typ (or a new floats.typ if preferred, but base.typ is cleaner since it applies globally). In `apply-base-styles`, add:
- `show figure.caption: it => { set text(size: 9.5pt); it }` — slightly smaller caption text
- This addresses TMPL-08 (figure and table caption styling). The fig_note() and tab_note() R helpers from Phase 2 emit markdown that appears below the figure/table; their Typst rendering is handled by the standard Pandoc-to-Typst pipeline. The caption show rule ensures consistent sizing.
