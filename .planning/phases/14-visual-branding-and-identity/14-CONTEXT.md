# Phase 14 Context: Visual Branding and Identity

## Goal
Establish a professional visual identity for the typstR documentation site by implementing a custom logo, favicon, and branded CSS accents.

## Requirements
- BRAND-01: Custom logo in navbar and matching favicon.
- BRAND-02: Branded CSS accent colors (links, sidebar highlights, callouts).

## Success Criteria
1. Custom logo visible in the navbar across all pages.
2. Favicon visible in the browser tab.
3. Branded accents (e.g., primary link color) are consistent across the site.
4. `pkgdown::check_pkgdown()` returns zero warnings.

## Constraints
- Use Bootstrap 5.
- Keep branding aligned with the "scientific/academic" aesthetic of typstR (clean, professional).
- Ensure `pkgdown/` directory remains ignored in the R package build (.Rbuildignore).
