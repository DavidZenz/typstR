    Create R/metadata_helpers.R with three exported functions and their print methods:

    1. **author(name, affiliation = NULL, email = NULL, orcid = NULL, corresponding = FALSE)**
       - Validate: name must be character(1). If orcid is non-NULL, validate with regex `^[0-9]{4}-[0-9]{4}-[0-9]{4}-[0-9]{3}[0-9X]$` — error via cli::cli_abort with message "Invalid ORCID format: {.val {orcid}}" and hint "Expected format: XXXX-XXXX-XXXX-XXXX".
       - If affiliation is non-NULL, coerce to character vector.
       - If corresponding is non-NULL, must be logical(1).
       - Return: structure(list(name=, affiliation=, email=, orcid=, corresponding=), class = c("typstR_author", "list"))
       - Remove NULL fields from the list before structuring (use Filter(Negate(is.null), ...)) so that yaml::as.yaml() does not emit `affiliation: ~` for unset fields.

    2. **affiliation(id, name, department = NULL, address = NULL, country = NULL)**
       - Validate: id must be character(1), name must be character(1).
       - Return: structure(Filter(Negate(is.null), list(id=, name=, department=, address=, country=)), class = c("typstR_affiliation", "list"))

    3. **manuscript_meta(authors, affiliations = list(), keywords = NULL, jel = NULL, acknowledgements = NULL, report_number = NULL, funding = NULL, data_availability = NULL, code_availability = NULL)**
       - Cross-validate: extract all affiliation IDs from authors (unlist lapply `[[` "affiliation"), compare against affiliation `id` fields. cli::cli_abort if dangling IDs found with message "Author references undefined affiliation IDs: {.val {dangling}}" and hint.
       - Build typstR block: Filter(Negate(is.null), list(keywords=, jel=, acknowledgements=, "report-number"=, funding=, "data-availability"=, "code-availability"=)). For keywords and jel, unclass() them if they are S3 objects.
       - Assemble result: list(author = lapply(authors, unclass), affiliations = lapply(affiliations, unclass)). If typstR block non-empty, add result[["typstR"]] <- typstR_block.
       - Return: structure(result, class = c("typstR_meta", "list"))

    4. **Print methods**: print.typstR_author prints `<typstR_author> {name}`, print.typstR_affiliation prints `<typstR_affiliation> [{id}] {name}`, print.typstR_meta prints `<typstR_meta> {N} author(s), {M} affiliation(s)`.

    All functions use roxygen2 with @export. Use cli::cli_abort for errors (not stop()). Follow the exact error handling style from R/create_working_paper.R.
