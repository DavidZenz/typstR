# typstR: Typst-Based Quarto Output Formats for Research Publications

Helpers for scaffolding and rendering working papers, articles, and
policy briefs with shipped Typst-based Quarto formats.

## Details

The package centers a YAML-driven workflow: scaffold a starter project,
edit `template.qmd`, and render a polished PDF without editing Typst
source.
